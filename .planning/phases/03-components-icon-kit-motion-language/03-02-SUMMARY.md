## Decisions Made
- **Deterministic generator over hand-retyping the 20 SVGs.** A one-off Node script read each source file and stripped only the three root attributes, guaranteeing byte-for-byte geometry fidelity and eliminating transcription risk. The engine file was edited in place by anchored insertion (never a blanket rewrite).
- **lantern-gold authored as craft, not a tint.** Recolour firm-mapped per the UI-SPEC table (`#2E6BF5`→`--gold`, `#2536CE`→`--gold-deep`, `#EAF0FE`→`#FFF3CC`, `#C9D7F5`→dark warm `#4A3610`), then night-register-safety applied: the hoop, hanger, cap bead, base finial and both sparkles (everything on the dark field) lifted to `--gold2` `#FFD34D`; the flame kept `--gold-deep` because it renders on the bright cream panel. A headless-Chrome render on the `#241A05→#3A2A08` gradient confirmed it reads intentional and holds contrast.
- **`AW.KIT['lantern-gold'] = …` bracket assignment** rather than an in-literal edit — matches the D-33 must-have's own reference and keeps the 20-scene literal untouched.
