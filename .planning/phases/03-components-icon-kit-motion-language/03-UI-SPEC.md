# Phase 3 ‚Äî UI Design Contract

> Visual and interaction contract for the shared component layer, icon registry, and motion
> language. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **This spec EXTENDS the Phase-1 foundation contract ‚Äî it never contradicts it.** Every
> spacing / type / colour / radius / shadow / motion value below is a Phase-1 token being
> *applied* to a component, not re-invented. The only new tokens this phase adds are four
> depth/overlay values (declared explicitly in "Phase-3 Token Additions"). Design direction is
> LOCKED: elevate Josh's Gen-3 warmth ‚Äî cream field, soft large radii, indigo-tinted shadows,
> gummy 3D press ‚Äî to a world-class bar. Do NOT redirect into generic minimal SaaS.

> learner-miss colour (never red); nothing celebratory (confetti / PERFECT / combo) ever renders
> over or adjacent to a scripture / `.ayah` block; the pending-review pill is calm and lowercase,
> never alarming.

## Phase-3 Token Additions

Four new custom properties added to the `@layer tokens { :root }` content block (a token *addition*, not a layer-order redeclare ‚Äî permitted). Keeps the `@layer components` rules literal-free (D-40 "no literal px/hex where a token exists").

| Token | Value | Purpose | Rationale |
|-------|-------|---------|-----------|
| `--press-rest` | `5px` | Gummy hard-shadow rest offset | Gen-3's proven `0 5px 0` button depth; tokenized per D-40 |
| `--press-active` | `2px` | Gummy hard-shadow pressed offset | Gen-3's `0 2px 0` (equals `--sp-2`); the press translate = `calc(var(--press-rest) - var(--press-active))` = 3px, so the drop derives from the two, no third literal |
| `--scrim` | `rgba(30,40,80,.42)` | Bottom-sheet backdrop overlay | Gen-3's sheet scrim value; a dark overlay tint the 60/30/10 palette doesn't carry, so it earns a token |
| `--overlay-hero` | `rgba(37,54,206,.94)` | PERFECT overlay field (brand indigo) | Gen-3's PERFECT scrim; the brand indigo (same hue as `--sh-*` shadow tint) at high opacity for the celebration overlay |

No other tokens are added. All confetti / gradient / combo colours reuse existing `--gold / --gold2 / --gold-deep / --flame / --flame2 / --accent-bright / --green` tokens.

## Icon Registry (FND-04)

**Single source:** `AW.KIT` (20 branded scene icons) + `AW.GLYPHS` (13 UI glyphs) + `lantern-gold` (authored variant), all inline strings authored at parse time in the engine's `KIT` banner section (D-22, D-31, D-34). Gen-3's embedded 12-icon KIT and `learn.html`'s duplicate `UIC` / `IC_*` / `LANTERNG` constants are **superseded and deleted** ‚Äî this registry is the only icon source going forward.

**Authoring transform (per source SVG, deterministic ‚Äî D-31, hand-authored, `svgo` skipped):**
1. Strip root `xmlns=""`, `width="240"`, `height="300"`. **Keep `viewBox="0 0 240 300"`** so CSS controls size without distorting the 240:300 (0.8) portrait aspect.
2. Store as a backtick template literal (no `\"` escaping), one per icon, preceded by a one-line naming comment.
3. `svgo` is **NOT run** (decision ‚Äî see Recorded Decisions): the id-collision audit found zero `id` / `url(#‚Ä¶)` / `defs` / gradients across all 20 files, so `prefixIds`/`cleanupIds` are unnecessary and `npx` is avoided entirely. Hand-authoring is fully reviewable.

**Registry keys (20 scenes):** `mosque` ¬∑ `carpet` ¬∑ `lantern` ¬∑ `lanterns` ¬∑ `crescent` ¬∑ `hijab` ¬∑ `man` ¬∑ `family` ¬∑ `prostration` ¬∑ `standing` ¬∑ `quran-stand` ¬∑ `beads` ¬∑ `kaaba` ¬∑ `dua` ¬∑ `dates` ¬∑ `compass` ¬∑ `ewer` ¬∑ `night` ¬∑ `pattern` ¬∑ `calendar` ‚Äî plus **`lantern-gold`** (authored variant, 21st keyed entry).

**Glyph keys (13, `AW.GLYPHS`, ~24√ó24 square):** `flame` ¬∑ `spark` ¬∑ `check` ¬∑ `star` ¬∑ `cite` ¬∑ `lamp` ¬∑ `lock` ¬∑ `chest` ¬∑ `trophy` ¬∑ `fact` ¬∑ `remember` ¬∑ `fard` ¬∑ `angle`.

**`AW.UNIT_ICON` mapping (preserve Gen-3 semantics):** u1 ‚Üí `compass` ¬∑ u2 ‚Üí `lanterns` ¬∑ u3 ‚Üí `kaaba` ¬∑ u4 ‚Üí `mosque`.

**Accessor `AW.icon(name, {size?, label?})` (D-32):**
- Default ‚Üí injects `aria-hidden="true" focusable="false"` (decorative).
- `label` ‚Üí injects `role="img" aria-label="{escaped label}"` (ACC-02 forward-compat; label HTML-escaped).
- Missing / malformed name ‚Üí returns `''` (renders nothing ‚Äî never a broken image or tofu; never throws).
- `size` ‚Üí sets `width` + `height:auto` on scene icons (viewBox preserves the 0.8 aspect); glyphs (square) accept a square size. Mechanism (attr vs inline-style vs `.kit-ill` wrapper) is executor discretion (D-32).

## Component Anatomy & States

Every tappable component shares the base `:focus-visible` treatment (inherited: `outline:3px solid var(--accent); outline-offset:2px; box-shadow:0 0 0 6px var(--accent-soft)`; `--gold2` variant on the night register; shown on keyboard focus only). States below list rest / hover / active / focus-visible / disabled.

### Citation chip ‚Äî `.cite` (ENG-06 / D-37)

- **Anatomy:** `inline-flex`, gap `--sp-1`; leading cite glyph (`AW.icon('cite')` at glyph-inline 14px) + label; padding `2px var(--sp-2s) 2px var(--sp-1)`; radius `--r-xs` (8px); `white-space:nowrap`; `vertical-align:baseline`; font `0.9em` Inter 600.
- **Rest:** text `--accent-ink`, bg `--accent-soft`, border 1px `--accent-line`, glyph `--accent-ink`.
- **Hover:** fill `--accent`, text + glyph `--accent-on`; transition `--dur-3 --ease-standard`.
- **Active (press):** inline-light press ‚Äî `translateY(1px)` over `--dur-2 --ease-press` (no hard shadow; inline chips can't carry a 5px bottom offset ‚Äî see Press Physics reconciliation).
- **Focus-visible:** base accent ring.
- **Disabled:** n/a (citations are always live).
- **Behaviour:** tap ‚Üí `AW.wire` ‚Üí `AW.sheetRef(cfg.refs, id)` opens the citation sheet.

### Term chip ‚Äî `.term`

- **Anatomy:** inline; `--accent` dotted underline (`text-decoration: underline dotted var(--accent)` or bottom border), colour `--ink`, cursor pointer.
- **Hover:** underline goes solid `--accent`.
- **Active:** `translateY(1px)` over `--dur-2 --ease-press`.
- **Focus-visible:** base accent ring.
- **Behaviour:** tap ‚Üí `AW.sheetTerm(cfg.terms, id)` opens the gloss sheet.

### Bottom-sheet primitive ‚Äî `AW.sheet(html)` (D-35)

- **Structure:** singleton lazily created on `<body>`: `.scrim` (fixed, `inset:0`, bg `--scrim`) wrapping `.sheet` (`role="dialog" aria-modal="true"`). One element reused ‚Äî opening replaces content and closes any open sheet.
- **Sheet box:** fixed, `left/right:0; bottom:0; margin-inline:auto; max-width:480px` (matches the shell column, so on desktop it centres like the column, not full-window); bg `--paper`; radius `var(--r-xl) var(--r-xl) 0 0` (24px top corners only); padding `var(--sp-6) var(--sp-6) var(--sp-8)`; box-shadow `--sh-3`; `max-height:82%`; `overflow-y:auto`.
- **Grip:** `.grip` 40√ó4px, `--line2`, radius pill, centered, `margin-bottom:--sp-4` (decorative affordance).
- **Close affordance (D-35):** `.sheet-x` button, top-right, 44√ó44 hit area, `aria-label="Close"`; also Escape closes; scrim/outside tap closes.
- **A11y now:** `role="dialog"` + `aria-modal="true"`; `invoker = document.activeElement` captured for focus-restore on close; `close()` idempotent. **Deferred to Phase 6:** focus-trap, `inert`/`aria-hidden` on background, full iOS position-fixed scroll lock. Structure (scrim wraps sheet) leaves room for the trap.
- **Scroll lock:** `.sheet-lock { overflow:hidden }` on `<html>` while open (file:// and http identical).
- **Reused by Phase 5 (LRN-06):** streak / noor / course-switcher sheets MUST use this same primitive ‚Äî no second sheet implementation.

**Sheet choreography (from the token set ‚Äî D-39):**

| Phase | Scrim | Sheet | Duration | Easing |
|-------|-------|-------|----------|--------|
| IN (open) | opacity 0 ‚Üí 1 | `translateY(100%)` ‚Üí `translateY(0)` | `--dur-4` (280ms) | `--ease-gentle` (soft settle) |
| OUT (close) | opacity 1 ‚Üí 0 | `translateY(0)` ‚Üí `translateY(100%)` | `--dur-4` | `--ease-in` (clean exit) |

Gentle in, clean out ‚Äî the calm, un-hurried sheet motion the whole app inherits.

### `AW.cite(id, label)` / `AW.wire(root, cfg)` (D-37)

## Press Physics (MOT-03 / D-40)

**Signature gummy press** ‚Äî one shared, token-defined pattern (shadow-collapse + `translateY` on `:active`), applied across the FULL tappable inventory.

**Elevated inventory** (`.btn`, `.opt`, `.tf`, `.tile`, `.tab`, `.hstat`, and Phase-5 `.node`):
- **Rest:** `box-shadow: 0 var(--press-rest) 0 var(--accent-deep)` (hard, non-blurred indigo/accent offset) + optional soft `--sh-1`.
- **Active:** `transform: translateY(calc(var(--press-rest) - var(--press-active)))` (= 3px) + `box-shadow: 0 var(--press-active) 0 var(--accent-deep)`.
- **Transition:** `transform var(--dur-2) var(--ease-press), box-shadow var(--dur-2) var(--ease-press)`.

**Inline chips** (`.cite`, `.term` ‚Äî reconciliation): inline elements can't carry a 5px bottom hard-shadow without breaking line layout, so they share the **same timing token** (`--dur-2 --ease-press`) with `translateY(1px)` and no hard shadow. This keeps the press *feel* coherent while respecting inline flow. Recorded so the checker reads it as intentional, not a gap.

**Sheet rows** (`.sheet-row`, Phase-5 course-switcher): elevated-lite ‚Äî `:active { transform: translateY(1px); }` + a soft `--accent-soft` background shift.

**Generic `.btn` state contract:**

| State | Treatment |
|-------|-----------|
| Rest | `--accent` fill ¬∑ `--accent-on` label ¬∑ `0 var(--press-rest) 0 var(--accent-deep)` |
| Hover (pointer) | fill `filter: brightness(1.03)` ‚Äî subtle; buttons are already 3D, no structural change |
| Active | gummy collapse (above) |
| Focus-visible | base accent ring (`--gold2` on night register) |
| Disabled | `opacity:.5; box-shadow:none; transform:none; pointer-events:none` (contract exists; v1 has few disabled states ‚Äî the un-loseable promise means most actions stay live) |

**Depth tokens:** `--press-rest` 5px / `--press-active` 2px; the 3px drop derives as `calc(rest ‚àí active)` ‚Äî no third literal (D-40 "no literal px where a token exists").

## Reduced Motion (MOT-04 / D-42)

One global quieting mechanism, defined once in `@layer motion`, with **two triggers sharing the same rule bodies**: `@media (prefers-reduced-motion: reduce)` (OS setting) AND `[data-motion="reduce"]` (the Phase-2 `awba_prefs` boot-stamp, which fires only on the in-app override ‚Äî so both are required to cover both users).

**Mechanism:**
- **Collapse** `--dur-1 ‚Ä¶ --dur-6` to `1ms` ‚Äî instantly quiets every CSS transition and every finite entrance (each component written once, inherits the collapse). State-communicating motion (press, sheet-in) thus **shortens, not vanishes** (D-42). Works because `@layer motion` is highest-priority, overriding the `tokens`-layer defaults.
- **`animation: none !important`** on the enumerated infinite loops (`.companion`, `.breathing-ring`, `[data-ambient]`) ‚Äî collapsing an infinite loop's duration to 1ms spins it at ~1ms (a seizure risk, the opposite of the goal); loops must be *stopped*, not sped. `--dur-amb` is deliberately NOT collapsed (the loop is stopped outright).
- **JS motion self-guards** at call time: `AW.confetti` no-ops; `AW.animate` sets `dur=1` (PERFECT reduces to a simple fade). `AW.reducedMotion()` = `matchMedia('(prefers-reduced-motion: reduce)').matches || documentElement[data-motion]==='reduce'`.

**Per-component reduced-motion behaviour (contract):**

| Component | Normal | Reduced |
|-----------|--------|---------|
| Gummy press | translate + shadow collapse over `--dur-2` | ~instant (1ms) ‚Äî still confirms the tap |
| Sheet IN / OUT | slide over `--dur-4` | ~instant appear / dismiss (shortened) |
| Node pop-in | overshoot spring `--dur-5` | instant place (no overshoot) |
| Combo chip | scale pop | instant show |
| PERFECT overlay | scale pop + gradient (WAAPI) | simple fade (`dur=1`) |
| Confetti | falling divs | **no-op** (nothing spawns) |
| Companion bob / glow | infinite loop | `animation:none` (still) |
| Breathing ring | infinite pulse | `animation:none` (static ring) |

**preview.html demonstrates BOTH trigger paths** ‚Äî the OS "Reduce motion" setting AND an in-page `data-motion="reduce"` toggle on `<html>` ‚Äî and the reviewer sees confetti + companion loop visibly stop under each.
