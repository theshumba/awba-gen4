## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | AW.S.load / JSON.parse of awba_state | mitigate | try/catch every getItem+JSON.parse; corrupted blob falls through to default v1, never throws (Pitfall 5) |
| T-02-02 | Denial of Service (self) | migrateFromLegacy per-field reads | mitigate | independent try/catch per legacy key; `.filter(isNaN)` drops unparseable dates; partial legacy set still migrates what is valid (Pitfall 6) |
| T-02-03 | Information Disclosure | localStorage plaintext (noor/stars/days) | accept | single-user device-local by design; nothing secret persisted (PROJECT.md) |
| T-02-04 | Tampering (XSS) | string-concat innerHTML in future runners | accept/forward | STATE section builds no DOM except the guarded boolean-attribute stamp; XSS is a Phase-4 runner concern — flagged forward, not solved here |
| T-02-SC | Tampering (supply chain) | npm/pip installs | not-applicable | zero external packages by D-25; Node core only (node:test/vm/assert/fs); no `package.json`, no install task, no registry surface to slopcheck |
