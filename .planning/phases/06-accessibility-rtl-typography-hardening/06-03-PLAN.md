The method (06-RESEARCH §Contrast Audit — computed styles from rendered pages): (1) TreeWalk `document.body` text nodes, skip empty/whitespace, skip `.aw-sr`/`aria-hidden`; for each `getComputedStyle(el)` → color, fontSize, fontWeight (large = >=24px or >=18.66px bold → 3:1, else 4.5:1). (2) Effective background = walk ancestors up, alpha-composite each non-transparent `backgroundColor` (srgb "over") onto the accumulator until opaque; if `<html>` reached with alpha remaining, composite onto the register ground token (read the register class → its token). (3) Grain overlay = a tiled PNG at ~2–9% opacity, invisible to computed backgroundColor → ACCEPTED approximation: ignore, document in the audit header. (4) Ratio = WCAG relative luminance (sRGB linearization) + `(L1+0.05)/(L2+0.05)`. (5) Non-text 3:1 (1.4.11): sweep `.opt`/`.tf`/`.tile` borders, focus-ring token pairs, thermal `[data-state]` shapes, `.np-seed`, timer bar — computed borderColor/outlineColor vs the same effective-bg. (6) Report `CONTRAST OK/FAIL <page> <selector> <fg> on <bg> = N.NN:1 (needs X)`; exit non-zero on FAIL.

§2.1 ground-truth numbers to reproduce (03-UI-SPEC-ATHAR §2.1): gold-on-Kiswah 8.40:1, ember-on-Kiswah 5.05:1 (OK); crimson-on-Kiswah 2.65:1, gold-on-cream 1.93:1, powder-on-cream 1.58:1, rose-on-cream 1.73:1 (BANNED as text — the audit proves no text node lands there). ember-on-cream 3.21:1 = large/shape only.

State-forcing table (06-RESEARCH §Contrast Audit — all verified feasible): fresh storage → u1m1 active + rest locked; a pre-IIFE seed `<script>` injected BEFORE the learn IIFE (`AW.S.set('stars',{...})` — engine loaded in `<head>`, AW exists) → done/stars/gold-thread/rosette/chest-available; `+ window.__awbaClaimChest('u1c')` → chest done + Festival; `.onode.click()` per node → popup variants; drive a real lesson: click a wrong `.opt` + `#check` (`.click()` works, synthetic Enter does not) → `.opt.wrong` + `.opt-why` + retry; click correct ×3 → correct/combo/flourish; drive a short lesson (or `scripts/fixtures/valid-lesson.html`) through verdict/noor/returns/done/ring/dua; `--virtual-time-budget=20000` on a review → `.low` + timeout mercy copy.
