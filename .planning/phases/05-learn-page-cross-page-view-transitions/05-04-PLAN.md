## Phase Goal

**As a** learner, **I want to** open Awba's front door and walk a beautiful winding path through the whole Aqeedah course — seeing where I am, what is next, and a daily verse — with calm page-to-page transitions, **so that** returning to learn feels like coming home to a living manuscript, not opening a game.

*(Vertical slice 3: after this plan tapping a node opens its slip-of-paper popup and START/REVIEW/LEGENDARY carries you to the lesson with a calm native morph — the path becomes walkable.)*

<objective>
Make the path interactive and the navigation alive: the singleton cream node popup (body-mounted, edge-clamped, arrow-offset, START/REVIEW/LEGENDARY CTAs + seed-row + stars, gentle locked microcopy), the cross-document View-Transitions opt-in inherited by all 20 pages, and the one shared-element `circuit-term` morph (tapped node square → lesson opener square) — degrading to a clean plain navigation over `file://`/Firefox with no fallback code.
